# Login-nuist  

需要环境：python  

使用方法：  

1. 确保`nuist-auto-login.py`和`setup.py`在同一文件夹下  
2. 在此文件夹路径中打开命令行，输入`python setup.py`，按照要求填写账号密码及运营商  
3. 成功生成`auto-login.vbs`后，双击执行即可  
4. 你也可以右击`auto-login.vbs`生成桌面快捷方式，从此连上`i-nuist`之后只需要双击快捷方式即可
